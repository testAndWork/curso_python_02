TArea de Arreglos en JS

1- Suma de Elementos:
    Crea un array de números.
    Escribe una función que calcule la suma de todos los elementos del array.

2- Filtrar Números Pares:
    Crea un array de números.
    Escribe una función que filtre los números pares del array y devuelva un nuevo array con esos números.

3- Encontrar el Número Mayor:
    Crea un array de números.
    Escribe una función que encuentre y devuelva el número más grande del array.

4- Contar Repeticiones:
    Crea un array de elementos (pueden ser números o strings).
    Escribe una función que cuente cuántas veces se repite cada elemento y muestre los resultados.

5- Eliminar Duplicados:
    Crea un array con elementos duplicados.
    Escribe una función que elimine los elementos duplicados y devuelva un nuevo array sin duplicados.

6- Ordenar Números:
    Crea un array de números desordenados.
    Escribe una función que ordene los números de menor a mayor.

7- Unir Arrays:
    Crea dos arrays.
    Escribe una función que una ambos arrays en uno solo y lo devuelva.

8- Buscar Elemento:
    Crea un array de elementos (pueden ser números o strings).
    Escribe una función que tome un elemento como parámetro y devuelva la posición en la que se encuentra en el array.

9- Invertir un Array:
    Crea un array.
    Escribe una función que invierta el orden de los elementos en el array y devuelva el nuevo array invertido.

10-Dividir Array en Grupos:
    Crea un array de elementos y escribe una función que tome un número como parámetro y divida el array en grupos del tamaño especificado, devolviendo un array bidimensional.
