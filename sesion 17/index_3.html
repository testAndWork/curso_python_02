Etiquta < embed > 


    El elemento < embed > se utiliza para incrustar contenido externo 
 en un documento HTML. El contenido externo puede ser una imagen, un 
 archivo de audio o video, o incluso una aplicación web.

El elemento < embed > es un elemento vacío, lo que significa que no
 tiene una etiqueta de cierre.
            
 El elemento < embed > tiene una serie de atributos que le permiten 
 controlar el aspecto y el comportamiento del contenido incrustado. 
 
 Los atributos más comunes son:
            
    src: Especifica la URL o la ruta del contenido externo.
    type: Especifica el tipo de contenido externo.
    width: Especifica el ancho del contenido incrustado.
    height: Especifica la altura del contenido incrustado.
    alt: Especifica un texto alternativo para el contenido incrustado.
            
 Para usar el elemento < embed >, debes crear una etiqueta < embed > 
 y especificar los atributos necesarios.
