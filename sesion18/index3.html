Tipos de etiquetas en form

Etiqueta tipo "text"

  • Crea el campo de texto en una linea
  • Acapta cualquier texto alfanumerico

Etiqueta tipo "password"

  • oculta el campo de texto

Etiqueta tipo "email"

  • Acepta y valida una direccion de correo electrónico

Etiqueta tipo "number"

  • Acepta numeros y ademas puede incluir atributos como min, max, step.

Etiqueta tipo "date"

  • Proporciona un selector de fechas

Etiqueta tipo "checkbox"

  • Crea una casilla de verificacion que permite al usuario seleccion en estilo opcion multiple

Etiqueta tipo "radio"

  • Crea una casilla de verificacion que permite al usuario seleccion en estilo opcion multiple

Etiqueta tipo "file"

  • Permite cargar un archivo

Etiqueta tipo "reset"

  • Borra oi reinicia el formulario
